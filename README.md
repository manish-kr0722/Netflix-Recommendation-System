# 🎬 Netflix Recommendation System

## 📌 Project Overview

With thousands of movies and TV shows available on streaming platforms, finding relevant content can be challenging. This project builds a personalized movie recommendation system using collaborative filtering to recommend movies based on user preferences and historical ratings.

The project also includes comprehensive Exploratory Data Analysis (EDA) to understand user behavior, rating patterns, and movie popularity.

---

## 🎯 Problem Statement

Streaming platforms generate massive amounts of user rating data. The objective of this project is to develop a recommendation engine that predicts movies users are likely to enjoy, improving user experience and engagement.

---

## 📂 Dataset

**Dataset:** Netflix Prize Dataset

The dataset contains:

- Customer IDs
- Movie IDs
- Ratings
- Rating Dates

The complete dataset contains over **24 million ratings**, making it suitable for large-scale recommendation systems.

---

## 🎯 Project Objectives

- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Analyze user rating behavior
- Identify popular and highly rated movies
- Build a recommendation model
- Generate personalized Top-N movie recommendations
- Evaluate model performance using RMSE

## 🤖 Machine Learning Model

### Algorithm

- Singular Value Decomposition (SVD)

### Why SVD?

SVD is one of the most widely used collaborative filtering algorithms for recommendation systems because it can accurately predict user preferences using latent factors extracted from historical ratings.

---

## 📈 Model Evaluation

Evaluation Metric:

- Root Mean Squared Error (RMSE)

The trained model was evaluated using cross-validation to measure prediction accuracy.
